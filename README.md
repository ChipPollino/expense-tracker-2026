# expense-tracker-2026

Репозиторий для выполнения индивидуального проекта по разработке прикладного веб-приложения для учета расходов.



Приложение предназначено для любого человека, желающего контролировать свои расходы и планировать бюджет.



Хранимые данные:

* Пользователи (login, пароль)
* Записи о расходах: сумма, дата, категория (еда, развлечения, медицина)
* Категории расходов (предустановленные и пользовательские)
* Пользовательские настройки (тема оформления, граница допустимых расходов, личные категории)



Основные действия:

* Добавление новой записи о расходе с указанием суммы, категории и даты
* Редактирование или удаление существующих записей
* Просмотр списка расходов с фильтрацией по дате, категории, сумме
* Просмотр статистики и отчетов: диаграммы по категориям, месячные итоги



Используемый стек:

* Язык: Python 3
* База данных: SQLite
* Backend: FastAPI
* Шаблоны (frontend): Jinja2
* HTTP-тестирование: Swagger/Postman
* Контроль версий: Git
* Контейнеризация: Docker
